What is EMV?

EMV is the chip card technology developed by Europay, Mastercard® and Visa®. Its technology offers more transaction security, lowers the chances for fraud and improves customer confidence.

Steps to prepare for the EMV update

  1. Be sure you have an EMV-capable terminal
    2. Your terminal should have both a swipe for magnetic stripe cards and a slot for inserting chip cards.

  2. Plan the time for the update
    3. Pick a slow day of the week, preferably after business hours.

  3. Close and batch out before you update even if you auto settle
    4. Settle transactions and run any end-of-day reports.

Best practices for running a pilot
If you operate multiple locations, install required EMV software and hardware in one store first to run a pilot.
Make sure staff is trained on chip card transactions.
Display EMV symbol to let cardholders know where they can make chip card payments.
Document problems and what went well in the pilot before rolling out to remaining locations.
